Materi: Cara Setting PK

Panduan konfigurasi PK pada sistem parkir dengan perintah terminal yang jelas dan rapi.

Langkah-langkah Konfigurasi PK

  • cd /home/client01/spsclient
  • sftp root@192.168.1.201 [ambil data ke PK lain]
  • cd /home/client01/spsclient
  • get
  • exit [2x untuk keluar sftp]
  • cd /home/client01/spsclient
  • vi Setting.ini [huruf S harus besar]
  • Ganti IP, simpan, dan exit
  • startx
  • Setting camera IN/OUT PK
  • cd /tmp
  • ls
  • mkdir bitmap_in
  • mkdir bitmap_out
  • service nfs restart
  • service netfs restart
  • service network restart
  • chmod 777
  • Exit atau reboot sistem

Catatan: Pastikan mengikuti urutan langkah dengan hati-hati agar PK dapat berfungsi normal dan sinkron dengan sistem PM.
