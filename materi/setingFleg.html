Setting EPS FLEG

ssh root@192.168.1.200

Terhubung ke EPS server sebagai root melalui SSH.

cd /opt

Masuk ke direktori /opt di server EPS.

sftp root@192.168.1.201

Membuka koneksi SFTP ke server lain untuk mengambil data EPS.

cd /opt

Pindah ke direktori /opt di server tujuan SFTP.

get -R epsclient

Mengambil folder epsclient beserta isinya dari server SFTP.

exit

Keluar dari koneksi SFTP.

chmod 777 -R epsclient

Memberi hak akses penuh ke folder epsclient agar bisa dijalankan.

cd /epsclient

Masuk ke folder epsclient untuk melihat file di dalamnya.

ls

Melihat isi folder epsclient.

cp Qt-5.12.12-nexel132.tar.gz /opt

Menyalin paket Qt ke folder /opt untuk instalasi.

tar -xzvf Qt-5.12.12-nexel132.tar.gz

Mengekstrak paket Qt di folder /opt.

vi epsfleg.ini

Mengedit konfigurasi EPS FLEG.

vi /root/.bash_login

Mengedit file login bash root untuk setting PATH atau environment.

rm /etc/system/timesyncd.conf

Menghapus file konfigurasi timesyncd agar bisa diatur ulang.

dpkg-reconfigure tzdata

Mengatur zona waktu server (Asia/Jakarta).

vi /etc/network/interfaces

Mengubah konfigurasi IP server EPS.

reboot

Merestart server agar semua konfigurasi diterapkan.

Catatan: Lakukan langkah-langkah dengan hati-hati. Salah satu perintah dapat mempengaruhi sistem EPS.
