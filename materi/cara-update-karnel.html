CARA UPDATE KERNEL

09 Oktober 2025 | 22:47 Kamis

1

cd /tmp/bitmap_in/update

Masuk ke direktori tempat file update kernel berada.

2

ls

Pastikan ada dua folder kernel di dalam direktori tersebut.

3

cd kernelf15

Masuk ke folder pertama kernel.

4

ls

Periksa isi folder, pastikan file RPM tersedia.

5

rpm -Uvh *.rpm

Lakukan update paket kernel dari folder pertama.

6

cd kernelfc19

Masuk ke folder kernel kedua.

7

ls

Cek kembali file RPM sebelum update.

8

rpm -Uvh *.rpm

Update kernel dari folder kedua.

9

reboot

Restart sistem untuk menerapkan kernel baru.

Catatan: Pastikan proses update berjalan tanpa error sebelum melakukan reboot.
