Setting IP Kamera untuk Sistem Parkir

Langkah-langkah konfigurasi kamera IP agar terhubung dengan server monitoring

1. Jenis Kamera yang Digunakan

  • Kamera IP (PoE atau non-PoE)
  • Merek umum: Hikvision, Dahua, UNV, atau OEM lainnya
  • Mendukung RTSP atau ONVIF

2. Menentukan IP Kamera

Gunakan software dari vendor (contoh: SADP Tool dari Hikvision) untuk mendeteksi kamera di jaringan.

  • Hubungkan kamera ke switch jaringan atau langsung ke laptop
  • Buka SADP Tool â†’ lihat IP default (misalnya 192.168.1.64)
  • Ubah IP agar sesuai dengan subnet server (contoh: 192.168.1.101)
  • Set gateway dan DNS jika perlu akses cloud

3. Login ke Kamera via Browser

  1. Masukkan IP kamera di browser (Chrome/Edge)
  2. Login (default user: admin, password: 12345 atau sesuai vendor)
  3. Set resolusi, framerate, dan stream RTSP jika digunakan oleh server
  4. Aktifkan protokol ONVIF jika menggunakan software pihak ketiga

4. Integrasi ke Server Monitoring

  • Gunakan software NVR seperti ZoneMinder, iVMS, VLC, atau CMS dari vendor
  • Tambahkan IP dan RTSP stream ke sistem
  • Contoh URL RTSP: rtsp://admin:password@192.168.1.101:554/h264
  • Uji koneksi dan tampilan video realtime

5. Tips Keamanan & Jaringan

  • Ganti password default kamera segera setelah aktivasi
  • Blok akses internet jika kamera hanya untuk lokal (via firewall)
  • Gunakan VLAN khusus untuk kamera jika sistem besar
