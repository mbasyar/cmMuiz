Cara Pasang Admin di POS

Langkah-langkah

cd /home/client01

Masuk ke direktori client POS.

sftp root@192.168.1.55

Koneksi ke server remote POS untuk mengambil data admin.

cd /home/adm01

Pindah ke direktori admin di server POS.

get -R spsadmin

Menyalin folder spsadmin secara rekursif ke lokal POS.

exit

Keluar dari sesi SFTP.

vi /etc/WindowMaker/WMRootMenu

Buka file menu root untuk menambahkan menu admin.

Tambahkan baris ke kedua:
("Admin", EXEC, "/home/client01/spsadmin/sps_225R129"),

startx

Mulai kembali antarmuka grafis POS.

F12

Cek apakah menu Admin muncul dan bisa dijalankan.
