Perintah Setelah Instal Ulang Fedora 30

Panduan konfigurasi awal Fedora 30 agar sistem siap digunakan dan sinkron dengan perangkat parkir.

Langkah-langkah Konfigurasi

systemctl disable firewalld.service

Menonaktifkan firewall Fedora agar tidak menghalangi koneksi internal sistem parkir.

systemctl enable httpd

Mengaktifkan layanan web server Apache agar dapat diakses secara otomatis saat boot.

systemctl enable sshd

Mengaktifkan layanan SSH untuk akses remote ke server.

vi /etc/auto.master

Membuka file konfigurasi auto-mount, hilangkan tanda pager jika muncul untuk menampilkan seluruh isi file.

vi /etc/auto.bitmap_in

Membuat atau mengedit konfigurasi auto-mount untuk folder bitmap_in.

vi /etc/auto.bitmap_out

Membuat atau mengedit konfigurasi auto-mount untuk folder bitmap_out.

showmount -e 192.168.1.50

Menampilkan daftar folder yang dibagikan oleh server NFS untuk memastikan koneksi tersedia.

systemctl enable autofs

Mengaktifkan layanan auto-mount agar folder remote dapat diakses otomatis saat boot.

systemctl restart autofs

Memulai ulang layanan auto-mount agar perubahan konfigurasi diterapkan.

cd /tmp/bitmap_in/data

Masuk ke folder sementara untuk menyimpan data bitmap yang akan digunakan oleh sistem parkir.

Catatan: Lakukan langkah-langkah ini secara berurutan agar Fedora 30 siap digunakan untuk sistem parkir dan tidak terjadi konflik layanan.
