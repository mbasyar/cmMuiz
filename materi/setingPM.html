Cara Setting PM

1. Login ke PM

  • telnet 192.168.1.101
  • Login root: quick sps login [root]

2. Konfigurasi Park.ini

  • cd /opt/quicksps
  • Lihat isi folder: ls
  • Edit konfigurasi: vi park.ini
  • Ubah: kelas kendaraan, nama PM, kode lokasi, dan pesan "Welcome to ..."
  • Simpan: :wq

3. Konfigurasi Net.conf

  • cd /etc
  • Edit net.conf sesuai PM: vi net.conf
  • Reboot PM: reboot

4. Sinkronisasi dengan PK

  • Telnet ke PK baru: telnet 192.168.1.201
  • Login root: root
  • Edit park.ini: vi park.ini dan salin db password dari PM lain
  • Edit fstab sesuai server: vi fstab
  • Simpan perubahan: :wq

5. Patch QuickSPS

  • Masuk ke folder image data: cd /opt/quicksps/image_in/data
  • Lihat versi PM: ls
  • Masuk ke folder versi PM: cd versi_pm
  • Jalankan patch: ./patch_quicksps
  • Reboot PM jika diperlukan
