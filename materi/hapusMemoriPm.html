Cara Hapus Memori PM

Panduan langkah-langkah untuk menghapus memori PM secara aman menggunakan terminal.

Langkah-langkah

telnet 192.168.1.106

Menghubungkan komputer ke PM melalui protokol telnet.

root

Login sebagai administrator PM agar dapat menghapus memori.

cd /sddisk

Masuk ke direktori tempat data PM disimpan.

ls

Melihat daftar file dan folder di direktori PM sebelum penghapusan.

rm -rf *

Menghapus semua file dan folder di direktori PM. Hati-hati! Perintah ini permanen.

Catatan: Pastikan semua data penting telah dicadangkan sebelum menjalankan perintah ini. Penghapusan memori PM bersifat permanen dan tidak dapat dikembalikan.
