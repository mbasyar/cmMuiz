SETTING ADMIN LANG

cd /home/adm01

Masuk ke direktori adm01.

sftp root@192.168.1.55

Terhubung ke server ADM untuk transfer data.

cd /home/adm01

Pastikan berada di direktori tujuan.

get -R spsadmin

Mengambil seluruh folder spsadmin dari server.

exit

Keluar dari sesi SFTP.

cd /home/adm01/spsadmin

Masuk ke folder spsadmin untuk pengaturan hak akses.

chmod 777 .

Memberikan hak akses penuh pada folder saat ini.

cd /home/adm01/spsadmin/lang

Masuk ke folder lang untuk pengaturan bahasa.

chmod 777 .

Memberikan hak akses penuh di folder lang.

cd /home/adm01/spsadmin/fiscord

Masuk ke folder fiscord.

chmod 777 .

Memberikan hak akses penuh di folder fiscord.

exit atau reboot

Keluar dari terminal atau restart sistem agar perubahan diterapkan.

Catatan: Pastikan folder spsadmin diambil dari server yang benar dan hak akses diterapkan dengan hati-hati.
