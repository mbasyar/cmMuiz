Instalasi Windows & Dual Boot

Digunakan untuk sistem hybrid antara Fedora (Server) dan Windows (Client Tools)

1. Kapan Perlu Windows?

Beberapa tools atau software hanya tersedia di Windows, seperti:

  • POS Client versi Windows
  • Tool konfigurasi IP kamera
  • VNC Viewer & TeamViewer untuk remote

2. Instalasi Windows

  1. Download ISO Windows 10 dari situs resmi
  2. Buat USB bootable dengan Rufus
  3. Boot dari USB → ikuti wizard instalasi
  4. Di bagian partisi, sisakan sebagian ruang untuk Fedora (misalnya 40GB)
  5. Instal Windows di partisi utama → reboot → selesaikan setup

3. Instal Fedora Setelah Windows

Agar bootloader Fedora bisa mengenali Windows, pastikan Fedora diinstal setelah Windows.

  • Boot USB Fedora
  • Pilih “Manual Partitioning” dan gunakan ruang kosong yang tersisa
  • Jangan hapus partisi Windows!
  • Lanjutkan instalasi Fedora seperti biasa
  • Pastikan GRUB bootloader terinstal di /dev/sda

4. Memilih OS Saat Booting

Saat menyalakan komputer akan muncul menu pilihan OS:

  • Fedora (default)
  • Windows Boot Manager

Ubah urutan default OS lewat terminal Fedora:

sudo grub2-set-default 1

5. Tips & Masalah Umum

  • Hindari instal Fedora duluan → Windows bisa menghapus GRUB
  • Selalu backup data sebelum membagi partisi
  • Jika menu boot tidak muncul: periksa BIOS (UEFI vs Legacy)
  • Gunakan aplikasi EasyBCD (Windows) jika GRUB bermasalah
