Cara Instalasi Windows

1 Siapkan USB bootable atau DVD Windows. Backup data penting.
2 Hidupkan komputer lalu tekan F2 / F10 / DEL / ESC untuk masuk BIOS/UEFI.
3 Cari menu Boot Priority / Boot Order. Set USB/DVD sebagai urutan pertama boot.
4 Simpan pengaturan dan restart komputer untuk boot dari USB/DVD.
5 Pilih bahasa, format waktu & keyboard, lalu klik Next → Install Now.
6 Masukkan Windows Product Key jika ada, atau klik "I don’t have a product key".
7 Pilih Custom (Install Windows only) untuk instalasi bersih.
8 Pilih partisi disk untuk instalasi. Hapus partisi lama jika ingin bersih → New → Apply → Next.
9 Tunggu proses instalasi selesai, komputer akan restart beberapa kali. Jangan cabut USB/DVD.
10 Konfigurasi awal: pilih wilayah, layout keyboard, buat akun pengguna & password, atur privasi.

Catatan: Pastikan semua data penting dicadangkan sebelum instalasi. Ikuti urutan langkah agar instalasi berjalan lancar.
