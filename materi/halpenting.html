HAL-HAL PENTING CARA SETTING DI FEDORA

Konsole / xterm / Terminal

Aplikasi yang memungkinkan kita menjalankan perintah di sistem Linux. Biasanya tampilannya diawali dengan [root@localhost root]#.

  • Admin → klik kanan lalu pilih New Terminal atau tekan Alt + F2, lalu ketik xterm.
  • SPS pos masuk → tekan Ctrl + Alt + F1, login root (password: 123456).
  • CPMS pos masuk–keluar → tekan F12 lalu pilih xterm atau tekan Ctrl + Alt + Backspace.

Restart Program atau Aplikasi

Digunakan bila ada perubahan pada database atau file konfigurasi di pos.

  • SPS pos masuk → tekan Ctrl + Alt + Backspace.
  • CPMS pos masuk–keluar → tekan Ctrl + Alt + Backspace, lalu ketik startx dan login kembali.

Restart System

Digunakan bila restart aplikasi tidak cukup atau ada perubahan sistem.

  • SPS pos masuk → tekan Ctrl + Alt + F1, login root, lalu ketik reboot atau Ctrl + Alt + Del.
  • CPMS pos masuk–keluar → tekan F12 → ReBoot, atau tekan Ctrl + Alt + Backspace lalu ketik reboot.

Restart File Sharing

Digunakan jika kamera atau sistem jaringan tidak tampil (khusus lokasi SPS atau CPMS dengan FLAZZ BCA).

service netfs restart

Pastikan muncul tulisan [OK] di sisi kanan. Jika failed, cari tahu penyebabnya.

Perintah-Perintah Dasar Linux

cd → berpindah direktori

cp → menyalin file atau folder (cp -R untuk folder)

ls → melihat isi direktori

rm → menghapus file (rm -rdf untuk folder)

mkdir → membuat folder baru

mv → memindahkan atau rename file

vi → mengedit file (gunakan Insert, Esc, lalu :wq atau :q!)

ssh root@ip → remote access

sftp root@ip → transfer file (gunakan get dan put)

Catatan: Simpan setiap perubahan konfigurasi dalam direktori backupsys.
