SETTING EPS FLEG

Langkah-langkah Setting EPS FLEG

ssh root@192.168.1.200

Mengakses server EPS FLEG melalui SSH dengan user root.

cd /opt

Masuk ke direktori /opt tempat biasanya software disimpan.

sftp root@192.168.1.201

Menghubungkan ke server lain untuk mengambil data EPS FLEG menggunakan SFTP.

cd /opt

Masuk ke direktori /opt di server tujuan.

get -R epsclient

Menyalin seluruh folder epsclient dari server tujuan ke server lokal.

exit

Keluar dari sesi SFTP setelah data berhasil diambil.

chmod 777 -R epsclient

Memberikan hak akses penuh ke folder epsclient dan semua isinya agar bisa dibaca/tulis/eksekusi.

cd /epsclient

Masuk ke folder EPS FLEG.

ls

Melihat daftar file dan folder di direktori saat ini.

cp Qt-5.12.12-nexel132.tar.gz /opt

Menyalin file Qt yang diperlukan ke direktori /opt.

tar -xzvf Qt-5.12.12-nexel132.tar.gz

Mengekstrak file Qt yang sudah disalin.

vi epsfleg.ini

Membuka file konfigurasi EPS FLEG untuk diubah sesuai kebutuhan.

vi /root/.bash_login

Membuka file konfigurasi login bash untuk sinkronisasi setting.

sftp root@192.168.1.201

Kembali ke server sumber untuk menyalin file konfigurasi login.

get .bash_login

Mengambil file .bash_login dari server sumber.

exit

Keluar dari sesi SFTP setelah file berhasil diambil.

rm -rf .bash_login

Menghapus file lama agar tidak konflik.

vi /root/.bash_login

Menyisipkan file login baru agar konfigurasi terbaru berlaku.

ls -la

Memeriksa semua file di direktori root beserta hak aksesnya.

Catatan: Ikuti urutan langkah dengan hati-hati agar EPS FLEG dapat berfungsi normal dan sinkron dengan sistem server.
