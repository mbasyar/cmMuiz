Studi Kasus Centralized Maintenance

Kumpulan permasalahan nyata di lapangan beserta solusi teknis oleh tim CM.

1. Sensor Loop Tidak Membaca Kendaraan

Masalah: Gerbang tidak terbuka meski mobil berhenti di area deteksi.

Solusi: Kabel loop diganti karena putus akibat gigitan tikus.

2. Barrier Gate Macet

Masalah: Palang tidak terangkat meski tiket sudah divalidasi.

Solusi: Cek kabel pastikan tidak ada yg longgar atau putus.

3. CCTV Area Pintu Masuk / Keluar Mati

Masalah: Rekaman kendaraan Masuk / keluar tidak tersedia.

Solusi: Cek kabel di Kamera pastikan semua tdk ada masalah

4. Mesin Tiket Tidak Mengeluarkan Tiket/Struk

Masalah: Pengendara tidak bisa masuk karena mesin tidak mengeluarkan tiket.

Solusi: Cek Printer dan kertas diganti baru.

5. Server Parkir Hang

Masalah: Sistem tidak merespons, transaksi macet.

Solusi: Server direstart, patch keamanan di-update, database dioptimasi.
